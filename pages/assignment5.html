Question: How do you prefer to implement event handling using pure JavaScript? What Js-Frameworks solutions have you tried on previous/current positions? Show some simple examples.

Answer

I always prefer binding event with the controls i.e. binding event with a button or textbox instead of separately adding event while scripting.
I have mainly worked on TIBCO General Interface where binding events were done using the plug and play mechanism where its the same way as binding event to the control. Same way is done in Angular JS as well.
Another way which I prefer using CSS. Examples of both the ways are shown below:
1. Event binding in HTML to the control. When you click on the submit button and alert is triggered.:

<input type="submit" value="Submit" onclick="alert('Hello World')">
2. Adding a hover to any text and that event is handled in the CSS file.
<label class="someClass">My name is Shaswata.</label>
Under css we write as 
.someClass {color:#333333;} 
.someClass:hover {color:#ff0000;}
So when someone hovers onto the text the colour changes to red and when mouse is moved out then it burns black.
